Базовые селекторы

  • Синтаксис: * ns|* *|*. Универсальный селектор.Выбирает все элементы.Пример: * будет соответствовать всем элементам на странице.
  • Синтаксис: .имяКласса.Этот селектор выбирает жлементы, основываясь на значении их атрибут "класс".Например: селектор index выбирет все элемнеты, с соответсвующим классом, который был определен в атрибуте clas=="index"
  • Синтаксис: #имяИдентификатора. Этот селектор выбирает элементы, которые основываются на значении id атрибута. Этот селектор должен быть уникальным и встречаться только один раз. Например: селектор #toc выберет элемент с иденфикатором toc ( который был определен в атрибуте id="toc")
  • Синтаксис: [attr] [attr=value][attr~=value][attr|=value] [attr^=value] [attr$=value][attr*=value] . Этот селектор выбирает все элементы, имеющие данный атрибут или атрибут с определенным значением. Пример:a[href*="book"] — все ссылки, название которых содержит book.
  • Синтаксис: элемент. Селектор элемент позволяет форматировать все элементы данного типа на всех страницах сайта. Например, h1 {font-family: Lobster, cursive;} задаст общий стиль форматирования всех заголовков h1.

    • В CSS для определения приоритета стилей мы используем механизм специфичности. Мы пишем каскады или сложные селекторы, повышаем вес селектора, таким образом указываем браузеру приоритетность одних над другими.
